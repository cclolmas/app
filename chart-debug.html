Guia de Depuração de Gráficos - CCLOLMAS

Este guia ajudará você a diagnosticar e resolver problemas de visualização de gráficos na plataforma CCLOLMAS.

Atalhos Úteis

Atalho Função
Ctrl + Shift + C Abrir painel de depuração de gráficos
Ctrl + Alt + D Mostrar diagnóstico de carregamento de dados

Parâmetros de URL

Adicione os seguintes parâmetros à URL para ativar recursos de depuração:

Parâmetro Função
?debug-charts Ativar modo de depuração detalhada dos gráficos
?diagnose-charts Mostrar automaticamente o painel de diagnóstico

Problemas Comuns e Soluções

1. Gráfico não aparece (container vazio)

Solução: Verifique se o container existe no DOM antes de tentar renderizar o gráfico. Use o painel de depuração para verificar se os containers estão sendo criados corretamente.

2. Dados não carregam (gráfico mostrando "Sem dados disponíveis")

Solução: Verifique se os dados estão sendo carregados corretamente. Use o diagnóstico de carregamento de dados para identificar falhas nas chamadas de API ou erros no processamento de dados.

3. Gráficos com erro de renderização

Solução: Verifique se há erros no console do navegador. O painel de depuração mostrará erros específicos para cada gráfico. Problemas comuns incluem formatação incorreta de dados ou conflitos entre bibliotecas.

Como Usar o Painel de Depuração

  1. Pressione Ctrl + Shift + C para abrir o painel
  2. Verifique o status de cada gráfico (inicializado, com erro, pendente)
  3. Para gráficos com erro, clique no botão "Corrigir" ou "Tentar Novamente"
  4. Use o botão "Recarregar Todos" para reinicializar todos os gráficos
Importante: Certifique-se de que todas as dependências de biblioteca (Chart.js, D3.js, etc.) estão carregadas antes de tentar renderizar os gráficos.

      backgroundColor: 'rgba(255, 99, 132, 0.8)',
    },
    // mais datasets...
  ]
}

    { name: 'Node 1' },

    // mais nós...
  ],
  links: [
    { source: 0, target: 1, value: 10 },
    // mais links...
  ]
}

Dica: Para casos mais complexos de depuração, adicione ?debug=true à URL e consute o console do navegador para ver mensagens detalhadas.
